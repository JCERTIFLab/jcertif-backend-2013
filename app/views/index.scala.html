JCertif Backend 2013

  • Liste des niveaux de partenariats
  • Ajouter une nouvelle categorie
  • Liste des categories
  • Supprimer une categorie
  • Liste des présentateurs
  • Ajouter un nouveau présentateur (Le contenu(body) de la requête est un nouveau présentateur représenté au format JSON)
  • Ajouter un status de session
  • Liste des status de session
  • Supprimer un status de session
  • Liste de toutes les sessions
  • Proposer une nouvelle session
  • Liste de tous les sponsors
  • Liste de tous les logins
  • Liste de tous les participants
  • Ajouter un nouveau participant
  • Changer le mot de passe d'un participant
  • Récupérer les informations d'un participant (GET /participant/get/{emailparticipant})
  • Mettre à jour toutes les informations d'un participant (le mot de passe est exclu)
  • Réinitialiser le mot de passe (POST /participant/{emailParticipant}/lostpassword)
  • Inscrire un participant à une session (POST /participant/{emailParticipant}/session/add/{idSession})
  • Désinscrire un participant à une session (POST /participant/{emailParticipant}/session/remove/{idSession})
  • Lister toutes les sessions d'un participant (GET /participant/{emailParticipant}/session/list)
  • Vérifier les informations de connection d'un participant
